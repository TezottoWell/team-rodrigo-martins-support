Perguntas Frequentes

Como faço para acessar meus treinos?

Após fazer login no aplicativo, seus treinos estarão disponíveis na tela principal. Selecione o dia desejado para ver os exercícios programados. Lembrando que o app faz parte de uma consultoria, e não é um app de academia. portanto somente quem adiquirir o plano poderá acessar os treinos.

Como faço para adiquirir o plano?

Para adiquirir o plano, basta ir em financeiro e entrar em contato com o Rodrigo Martins que é o Personal Trainer responsável pela sua consultoria.

Como funciona o contador de água?

O contador de água permite registrar sua ingestão diária. Basta clicar no botão "+" e adicionar a quantidade consumida em ml.

Como visualizar os vídeos dos exercícios?

Em cada exercício, há um ícone de vídeo que, ao ser clicado, abrirá a demonstração do movimento no YouTube.

Como funciona a contagem de passos?

O aplicativo utiliza os sensores do seu iPhone para contar os passos automaticamente. Certifique-se de permitir o acesso ao HealthKit quando solicitado.

Como excluir minha conta?

O aplicativo contém uma aba de gerenciamento de conta na tela principal após login, voce pode rolar até o final que encontra o botão 'Excluir minha conta', apos a confirmação todos os seus dados serão excluidos e não haverá reversão.

Problemas Comuns

O app não está contando meus passos

Verifique se:

  • As permissões do HealthKit estão ativadas
  • O iPhone está com você durante as caminhadas
  • O app está atualizado para a última versão

Problemas com Face ID

Se estiver tendo problemas com o Face ID:

  • Verifique se o Face ID está configurado no seu iPhone
  • Certifique-se de que concedeu permissão ao app para usar o Face ID
  • Tente remover e adicionar novamente seu Face ID nas configurações do iPhone
